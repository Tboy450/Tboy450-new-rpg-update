# Archived Future NPC Assets

Older NPC concept sprites moved out of the active future asset shelf.

## Files

- `ember_blacksmith_placeholder_before_import.png`: tiny generated placeholder
  Blacksmith concept that used to live at
  `assets/processed/future_assets/npcs/ember_blacksmith.png`. It was replaced
  by the larger user-supplied imported Blacksmith cutout and kept here only as
  visual history.

## Beginner Rule

Do not load these files in the active game. Use active gameplay sprites from
`assets/processed/npcs/` and inactive future concepts from
`assets/processed/future_assets/npcs/`.
