# Archived Future Assets

Older inactive concept assets that were replaced by better imported art.

## Folders

- `npcs/`: old NPC concept placeholders kept after newer imported NPC sprites
  replaced them.

## Beginner Rule

This folder is only for history. Future asset concepts that might still be used
later belong in `assets/processed/future_assets/`. Active gameplay sprites
belong in folders like `assets/processed/npcs/`, `assets/processed/enemies/`,
or `assets/processed/scenery/`.
