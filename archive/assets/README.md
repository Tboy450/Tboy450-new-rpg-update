# Archived Assets

`archive/assets/` stores imported or generated art that is not part of the
active game anymore.

## Beginner Rule

Files here are reference material. The active game should load art from
`assets/processed/`, not from `archive/assets/`.

## Folders

- `future_assets/`: older inactive concept sprites kept after newer imported
  versions replaced them.
- `ui/`: archived UI/title-screen images that were tested but removed from the active runtime.
