# Future Asset Pack

This folder holds transparent PNG assets that are ready for future experiments
but are **not loaded by the game yet**.

Beginner note:

- These files are safe to browse, rename, replace, or use as references.
- Do not assume an asset is active just because it is in this folder.
- To make one active later, move or copy it into the correct active folder
  (`equipment/`, `enemies/`, `npcs/`, `ui/`, or `effects/`) and then update the
  matching data/code reference.
- No file in this folder is currently referenced by `systems/assets.py`,
  `game_data/equipment.py`, `game_data/enemies.py`, or `game_data/story.py`.

## Style Notes

These assets use a simple transparent pixel-art style meant to sit near the
current Dragon's Lair RPG look:

- dark outlines
- high-contrast fantasy colors
- small glow accents
- simple readable silhouettes
- dragon, healer, ghost, knight, and sage-inspired motifs

## Folders

- `gear/`: unused future weapons, shields, and magic focuses.
- `scenery/`: unused future map props and tile-like objects.
- `enemies/`: unused future enemy sprite ideas.
- `npcs/`: unused future town or quest NPC sprite ideas.
- `characters/`: unused future player-class or companion sprite ideas.
- `attacks/`: unused future attack/effect cutout ideas.
