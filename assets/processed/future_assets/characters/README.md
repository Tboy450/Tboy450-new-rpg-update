# Future Character Assets

These transparent PNGs are unused future player-class or companion concepts.

- `future_warrior_idle.png`: alternate Warrior idle concept.
- `future_mage_idle.png`: alternate Mage idle concept.
- `future_rogue_idle.png`: alternate Rogue idle concept.
- `future_healer_idle.png`: future healer/support class concept.

To activate one later, move or copy it into `assets/processed/characters/`,
then update the active character sprite mapping in `systems/assets.py`.
