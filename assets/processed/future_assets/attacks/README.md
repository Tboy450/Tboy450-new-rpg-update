# Future Attack Assets

These transparent PNGs are unused future attack/effect concepts.

- `warrior_lion_cleave.png`: Warrior slash arc concept.
- `warrior_drake_guard_impact.png`: Warrior impact/burst concept.
- `mage_firestorm_burst.png`: Mage fire blast concept.
- `mage_teal_heal_sigil.png`: Mage or healer sigil concept.
- `rogue_shadow_crosscut.png`: Rogue cross-slash concept.
- `rogue_ghoststep_smoke.png`: Rogue smoke-step concept.

To activate one later, move or copy the PNG into an active effect folder under
`assets/processed/effects/`, then add a loader path in `systems/assets.py` and
draw it from the battle animation code that owns that move.
