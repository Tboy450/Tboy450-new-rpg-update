# Future Enemy Assets

These transparent PNGs are unused future enemy concepts.

Beginner note:

- `plague_knight.png`, `crystal_bat.png`, and
  `ember_drake_hatchling.png` are imported cutouts from a user-supplied
  green-screen sheet.
- The other small PNGs are older generated placeholder concepts and can be
  replaced later as better imported art arrives.

- `bone_mender_specter.png`: a medical-symbol ghost enemy concept.
- `ashbound_imp.png`: a fire/volcano small enemy concept.
- `crystal_bat.png`: imported crystal bat enemy cut from `assets/source/enemies/plague_bat_drake_sheet.png`.
- `plague_knight.png`: imported plague knight enemy cut from `assets/source/enemies/plague_bat_drake_sheet.png`.
- `masked_stalker.png`: a Ghost Face-inspired stalker enemy concept.
- `rune_golem.png`: a crystal/rune tank enemy concept.
- `ember_drake_hatchling.png`: imported ember drake hatchling cut from `assets/source/enemies/plague_bat_drake_sheet.png`.

To activate one later, move or copy it into `assets/processed/enemies/`, then
add the enemy data and sprite path references in the active game files.
