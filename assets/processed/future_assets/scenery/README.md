# Future Scenery Assets

These transparent PNGs are unused future map or town props.

- `healer_spring_tile.png`: a healing spring tile with medical-cross styling.
- `dragon_bone_gate.png`: a dragon-bone gateway for a dangerous map entrance.
- `ash_lantern.png`: a small fire lantern prop for town, caves, or volcano paths.
- `town_banner_post.png`: a banner/signpost prop for town population scenes.
- `triage_tent.png`: a field-care tent/map prop concept.
- `ribbon_waystone.png`: a route marker with medical-ribbon styling.
- `dragon_fire_brazier.png`: a dragon-fire brazier or map hazard prop concept.

To activate one later, place it in the correct active scenery folder or add a
new active scenery folder, then update the drawing code that owns that map area.
