# Future Outdoor Town Scenery Assets

These transparent PNGs are unused future props and building fronts for town map
work. They are meant for paths, plazas, shops, taverns, clinics, town hall areas,
and outdoor NPC scenes.

Beginner note:

- These are sprites, not Python drawings.
- They are not active in the game yet.
- A building front can be used as a map decoration first, then later become an
  entrance if the game adds indoor zones.

## Files

- `tavern_front.png`: tavern exterior with sign, windows, and warm colors.
- `shop_front.png`: general goods shop exterior with striped sign colors.
- `town_hall_front.png`: stone town hall facade with banner and crest.
- `healer_clinic_front.png`: healer clinic exterior with medical-cross sign.
- `blacksmith_stall.png`: outdoor forge stall with anvil and fire.
- `market_canopy.png`: striped market stand with produce/wares.
- `stone_well.png`: town well prop with water and wooden crank.
- `notice_board.png`: quest notice board with pinned papers.
- `lamp_post.png`: double lantern post for streets or town corners.
- `flower_planter.png`: small flower planter for soft town decoration.
- `crate_stack.png`: stacked crates for shop, alley, market, or dock dressing.
- `barrel_stack.png`: stacked barrels for tavern, storage, or shop dressing.
- `cobblestone_patch.png`: ground patch to help dress future town paths.
- `lion_fountain.png`: lion-themed fountain matching the Lion Sage motif.

## Generation

These sprites are generated by `scripts/generate_future_town_assets.py`. Edit
that script and rerun it if you want to regenerate the whole outdoor set with a
consistent style.
