# Future Indoor Scenery Assets

These transparent PNGs are unused future props for interiors such as taverns,
shops, town hall rooms, homes, clinics, and quest buildings.

Beginner note:

- These are sprites, not Python drawings.
- They are not active in the game yet.
- To use one later, load it with the same image-loading pattern used by the
  active character, enemy, equipment, or effect sprites.

## Files

- `bookshelf_full.png`: full bookshelf for homes, shops, town hall, or study rooms.
- `potion_shelf.png`: potion shelf for healer, mage, clinic, or shop interiors.
- `tavern_bar_counter.png`: long bar counter with bottles and mugs.
- `shop_counter.png`: small shop counter with coins and displayed goods.
- `scroll_desk.png`: writing desk with scroll and quill for town hall or quest NPCs.
- `town_hall_dais.png`: raised meeting platform for a mayor, guard captain, or council.
- `round_tavern_table.png`: small table for taverns, inns, or casual NPC seating.
- `inn_bed.png`: bed prop for inn rooms or home interiors.
- `weapon_rack.png`: weapon rack for blacksmith, guard, or equipment shop interiors.
- `armor_stand.png`: armor display for town hall, guard house, or armor shop scenes.
- `hearth_fireplace.png`: lit fireplace for taverns, homes, or town hall rooms.
- `medical_ribbon_rug.png`: healer-themed rug with a cross/ribbon motif.

## Generation

These sprites are generated by `scripts/generate_future_town_assets.py`. Edit
that script and rerun it if you want to regenerate the whole indoor set with a
consistent style.
