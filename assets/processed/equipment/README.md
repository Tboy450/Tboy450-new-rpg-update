# Processed Equipment Icons

This folder stores game-ready PNG icons for weapons, armor, and accessories.

Beginner note:

- Equipment data lives in `game_data/equipment.py`.
- Each equipment record has an `icon` filename.
- The icon filename should match a PNG in this folder.
- The Inventory screen loads these through `systems/assets.py`.

These icons are intentionally simple pixel-style placeholders. They give the
equipment system real assets now, while leaving room to replace individual
icons with hand-drawn or imported art later.
