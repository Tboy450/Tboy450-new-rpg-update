# Active Food Stall Scenery

These transparent PNGs are active Food Stall assets.

Beginner note:

- The Food Stall was previously drawn only with Python rectangles.
- The active town map now loads this canopy sprite first.
- The older Python stall drawing remains in `main.py` as a fallback.

## Files

- `food_stall_canopy.png`: outdoor striped food stall canopy and counter used on the town map.

## Active Data References

- Outdoor reference: `TOWN_BUILDINGS` stall record in `game_data/town.py`.
