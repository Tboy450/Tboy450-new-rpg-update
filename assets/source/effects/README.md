# Source Effect Assets

Raw GIF/video effect uploads before they are extracted into PNG frames.

## Files

- `flame_tornado_attack.gif`: raw player SPECIAL travel animation source.
- `fire_blast_impact.gif`: raw Mage Fire Blast impact source.
- `mage_magic_fireball.gif`: raw Mage normal MAGIC projectile source.

## Game-Ready Output

Extracted frames live under `assets/processed/effects/<effect_name>/` as
`frame_00.png`, `frame_01.png`, and so on.
