# Source NPC Assets

Raw or generated source images for friendly/story NPCs.

## Files

- `lion_sage_generated_chroma.png`: generated Lion Sage source image on a removable chroma-key background.
- `town_guard_generated_chroma.png`: generated town guard source image on a removable chroma-key background.
- `blacksmith_innkeeper_sheet.png`: user-supplied green-screen sheet with the
  blacksmith on the left and innkeeper/shopkeeper on the right.

## Game-Ready Output

Processed transparent PNGs live in `assets/processed/npcs/`.
Story placement and dialogue data lives in `game_data/story.py`.

Future inactive NPC cutouts from `blacksmith_innkeeper_sheet.png` live in
`assets/processed/future_assets/npcs/`.
