# Build Output Folder

`bin/` is where local Android builds place generated APK files.

## Expected Files After A Successful Build

- `dragons-lair-rpg-android.apk`: signed release APK copied to a stable filename.
- `dragons-lair-rpg-android-debug.apk`: compatibility copy kept for older links and notes.

## Beginner Rule

Do not edit source code here. If this folder is empty, no local APK build has
completed in this checkout yet. Build outputs are generated by
`scripts/build_android.sh` and by the GitHub Actions workflow.
